# Memorabilia
Web application that retrieves information about tv shows and movies. This system is fetching data from [The Movie Database API](https://developer.themoviedb.org/reference/intro/getting-started).

https://memorabilia.onrender.com

## Tech stack
* [HTML](https://developer.mozilla.org/es/docs/Web/HTML)
* [CSS](https://developer.mozilla.org/es/docs/Web/CSS)
* [Javascript](https://developer.mozilla.org/es/docs/Web/JavaScript)
* [React](https://es.react.dev/)
* [Vite](https://vite.dev/)
